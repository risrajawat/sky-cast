# 🌤️ SkyCast

SkyCast is a modern weather web application that allows users to search for any city and get real-time weather information instantly.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Displays temperature, humidity, and wind speed
* ☁️ Shows current weather conditions
* ⚡ Fast and responsive UI

---

## 🌐 API Used

This project uses the OpenWeatherMap API to fetch real-time weather data.

---

## 🛠️ Technologies Used

* React.js
* Tailwind CSS
* OpenWeatherMap API

---

## 📦 Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/risrajawat/sky-cast.git
   ```

2. Navigate to the project folder:

   ```bash
   cd sky-cast
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```
